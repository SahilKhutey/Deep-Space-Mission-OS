# Deep Space Benchmarks

Verification and validation (V&V) regression benchmarks for interplanetary trajectories and propulsion engines.

## Directory Structure

*   `nasa/` — Standard benchmark test cases compared directly with NASA JPL Horizon orbits.
*   `verification_reports/` — Automated test generation reports validating numerical solver accuracy limits (e.g. step sizes, energy conservation).

## Reference Cases

*   **Vallado Ex. 2-3**: Kepler equation solver validation.
*   **Vallado Ex. 8-3**: Lambert orbital transfer validation.
