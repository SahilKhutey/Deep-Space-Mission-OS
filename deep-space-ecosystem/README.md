## Architecture Flow

```
Scientific Knowledge (deep-space-knowledge)
        ↓
Datasets & Benchmarks (deep-space-datasets, deep-space-benchmarks)
        ↓
Scientific Core (deep-space-core)
        ↓
SDK & Tooling (deep-space-sdk)
        ↓

        ↓

```
