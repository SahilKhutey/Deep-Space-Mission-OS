# AI for Space Systems

## Mission Design AI
- Trajectory generation (RNN/LSTM)
- Architecture selection (reinforcement learning)
- Multi-objective optimization (NSGA-II + ML surrogate)

## Operations AI
- Anomaly detection (autoencoders, isolation forest)
- Health monitoring (LSTM, transformer)
- Autonomous decision-making (RL, MPC)

## Digital Twin AI
- Failure prediction (RUL — Remaining Useful Life)
- Surrogate modeling (neural ODEs)
- Inverse problems (parameter estimation)

## Key Constraints
- Radiation tolerance for onboard AI
- Real-time inference (< 1 Hz)
- Power budget for compute
- Verification & validation for ML
- Onboard vs ground-based split
