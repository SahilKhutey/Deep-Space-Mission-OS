# Earth → Lunar Missions

## Phase Sequence
1. Launch → LEO
2. LEO → Parking orbit
3. Parking → Trans-Lunar Injection (TLI)
4. TLI → Coast (3 days)
5. Coast → Lunar Orbit Insertion (LOI)
6. LOI → Lunar orbit
7. Lunar orbit → Descent
8. Descent → Landing

## Typical ΔV
| Phase | ΔV (m/s) |
|-------|----------|
| Surface → LEO | 9400 |
| TLI | 3200 |
| LOI | 900 |
| Descent | 1800 |
| **Total** | **~15,000** |

## Key Equations
Trans-Lunar orbit: hyperbolic escape with C3 ≈ −2 km²/s²
LOI: v_arr − v_circ at perilune
