# Guidance, Navigation and Control

## Subsystems

### Guidance
"Where do I need to go?"
- Reference trajectory generation
- Target selection
- Maneuver planning

### Navigation
"Where am I?"
- Orbit determination
- Attitude determination
- State estimation (Kalman filters)
- Sensors: star trackers, IMUs, GPS, deep-space transponders

### Control
"How do I get there?"
- Actuators: reaction wheels, thrusters, CMGs
- Control laws: PID, LQR, MPC
- Pointing accuracy: arcseconds
