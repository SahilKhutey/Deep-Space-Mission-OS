# Linear Algebra for Aerospace

## Matrices
Used in:
- Coordinate transformations (rotation matrices)
- State transition matrices (STM)
- Covariance propagation

## Eigenvalues & Eigenvectors
Av = λv

Used in:
- Principal axes of inertia tensor
- Stability analysis
- Modal decomposition
- Principal component analysis (PCA)

## Singular Value Decomposition (SVD)
A = UΣVᵀ

Used in:
- Low-thrust trajectory optimization
- Dimensionality reduction in mission telemetry
- Sensor calibration

## Coordinate Transformations
R ∈ SO(3): rotation matrices
T ∈ SE(3): rigid body transformations

Used in:
- ECI ↔ ECEF conversion
- Orbital plane ↔ inertial frame
- Spacecraft attitude representation

## Tensors
Used in:
- Inertia tensor
- Stress-strain
- Electromagnetic field tensor
