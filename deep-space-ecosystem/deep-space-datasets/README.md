# Deep Space Datasets

This repository houses planetary ephemerides, thruster test sheets, and physical reference constants for the Deep Space Ecosystem.

## Directory Structure

*   `planetary_ephemerides/` — Contains analytical and high-precision tabular ephemerides for solar system planets and minor bodies.
*   `propulsion_data/` — Contains experimental and manufacturer-provided test sheets for bipropellant, ion, and nuclear engines.

## Ephemerides References

Planetary state parameters are based on the J2000 epoch and validated against NASA JPL DE405/DE430 planetary ephemeris tables.
